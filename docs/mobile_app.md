# Mobile Application

## Flutter and Firebase Integration

The project was connected to a mobile application prototype using Flutter for the frontend and Firebase for app-side integration.

## Scan and Upload Flow

The mobile workflow supported capturing or uploading tomato leaf images for disease classification.

## Prediction Display

The app displayed the predicted disease class, confidence score, and the final accept/reject status produced by the inference pipeline.

## Greenhouse Testing

The mobile application was intended for greenhouse testing, where users could quickly check tomato leaf images and receive model feedback.
